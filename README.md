# Pengembangan Aplikasi Mobile Project
PENJELASAN FITUR-FITUR APLIKASI PREDIKSI KESEHATAN MENTAL BERBASIS ANDROID:
- yang pertama itu ada bagian halaman utama ketika klik mulai sekarang akan beralih kehalam home, screaning, konseling, artikel, dan profil 
- setelah masuk kehalaman ini yang pertama ada halaman home yang berisi : bagaimana perasaanmu hari ini dengan menampilkan emote, dan ada juga untuk tes mental kita dengan menjawab pertanyaan-pertanyaan dari aplikasi tersebut setelah menjawab pertanyaannya akan menampilkan    hasilnya, dan ada juga fitur akses cepat yang berisi meditasi cepat hasil tes dan jurnal harian.
- setelah itu halaman meditasi jadi pada halaman ini berisi macam-macam meditasi dari meditasi ketika tidur, stress, dan fokus dan ada lagu/heningan untuk kita melakukan meditasi.
- halaman konseling ini berisi untuk memudahkan kita konsultasi dan bertanya tentang kesehatan mental kita pada halam ini kami menyediakan konsultasi dab bertanya kepada AI dan Dokter.
- halaman artikel ini berisi tentang artikel-artikel beserta penjelasanya untuk kita baca.
- yang terakhir terdapat halaman profil ini berisi tentang data diri kita.
